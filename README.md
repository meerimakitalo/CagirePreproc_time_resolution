# <img src="https://ffortin-sci-edu.github.io/pictures/CAGIRE_fake_logo.png" width="20%">  Pre-processing Software with an increased time resolution

This is a modified version of the COLIBRI/CAGIRE Preprocessing Software (preproc). The original preproc can be found from [here](https://github.com/COLIBRI-CAGIRE/CagirePreproc/tree/main). In this version, it is possible to process the images in batches in order to study the temporal evolution of the magnitude of the GRB afterglow. This is the first version of the preproc with an increased time resolution and further testing and development is still required.

CAGIRE is a near-infrared imager (J, H) that will be mounted on the COLIBRI telescope (San Pedro Martir, Mexico).
As a part of the ground segment of SVOM, CAGIRE is designed to perform systematic followups of gamma-ray burst afterglows.


A complete description of CAGIRE's specifications and performance is available in [Nouvel de la Fleche et al. (2023)](https://ui.adsabs.harvard.edu/abs/2023ExA...tmp...39N/abstract).
